# REDZED Store V2
Mobile friendly REDZED garments catalog with Supabase database and admin panel.

Files:
- index.html: customer website
- admin.html: admin panel
- config.js: Supabase URL, anon key, WhatsApp number, admin PIN

Setup:
1. Upload all files to GitHub.
2. In Supabase Project Settings > API, copy Project URL and anon public key.
3. Edit config.js and paste values.
4. Open admin.html, login with ADMIN_PIN, add products.
